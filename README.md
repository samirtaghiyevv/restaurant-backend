# 🍽️ Restaurant Backend API

Bu layihə restoranlar üçün hazırlanmış **tam funksional backend API**-dir.  
Mobil tətbiq (React Native, Flutter) və ya web frontend (React, Vue və s.) ilə
birbaşa inteqrasiya üçün nəzərdə tutulub.

---

## 🚀 Tech Stack

- Node.js
- TypeScript
- Express.js
- PostgreSQL
- Socket.io (real-time)
- JWT Authentication

---

## 📦 Features

- JWT-based authentication (User / Admin)
- Product & category management
- Order creation with database transaction
- Real-time order status updates (Socket.io)
- Admin order management
- User order history
- Ready for mobile & web apps
